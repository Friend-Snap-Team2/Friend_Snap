## Prerequisites
Node.js must be installed: https://nodejs.org/en/download

## Team Setup Guide

1. Clone the repo
```powershell
git clone https://github.com/yourname/Friend_Snap.git
```

2. Navigate into the folder
```powershell
cd Friend_Snap
```

3. Install all packages
```powershell
npm install
```

4. Add the `.env` file sent to you privately — do **not** use `.env.example`

5. Start the server
```powershell
npm start
```

You should see `Connected to MongoDB!` in the terminal confirming everything is working.
